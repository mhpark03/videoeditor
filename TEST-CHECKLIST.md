## ⚡ 빠른 시작

```bash
# 1. 의존성 설치
cd video-editor
npm install

# 2. 백엔드 실행 (API 테스트용)
cd backend
SPRING_PROFILES_ACTIVE=local ./gradlew bootRun

# 3. 에디터 실행
cd video-editor
npm start
```

## 🔗 백엔드 연동 테스트 (10분)

### ✅ 1. 연결 확인
- [ ] 백엔드 실행 중 (`http://localhost:8080`)
- [ ] `renderer/api.js` URL 확인

### ✅ 2. AI 콘텐츠 업로드
- [ ] TTS → 백엔드 (AUDIO)
- [ ] Imagen → 백엔드 (IMAGE)
- [ ] Veo → 백엔드 (VIDEO)

### ✅ 3. 대시보드 확인
```bash
# firstapp 실행 후 확인
http://localhost:5173/audios  # TTS 음성
http://localhost:5173/images  # Imagen 이미지
http://localhost:5173/videos  # Veo 비디오
```

### ✅ 4. 데이터베이스 확인
```sql
SELECT id, title, media_type, video_type
FROM videos
WHERE video_type = 'AI_GENERATED'
ORDER BY created_at DESC
LIMIT 5;
```
